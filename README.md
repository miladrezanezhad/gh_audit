# GitHub Security Auditor

A professional CLI tool for automated GitHub security auditing with auto-fix capabilities, multi-organization scanning, and interactive HTML/JSON reporting.

## Features

- 🔑 **Secret Scanning**: Detect exposed secrets across all repositories and commit history
- 📦 **Dependency Scanning**: Find CVE vulnerabilities in your dependencies
- 🔧 **Security Configuration Audit**: Check branch protection, 2FA, security features
- 🤖 **Auto-Fix**: Automatically fix common security issues
- 📊 **Interactive Reports**: Beautiful HTML dashboard with charts and timeline
- ⚡ **Parallel Scanning**: Fast scanning across multiple organizations
- 🎯 **Zero False Positives**: Smart filtering and .auditignore support
